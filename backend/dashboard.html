Chrome Extension Setup (v2.0)

  1. Make sure the backend is running (green status above) with Ollama at localhost:12434.
  2. Open Chrome → chrome://extensions/ → enable Developer mode.
  3. Click Load unpacked → select the extension/ folder from this project.
  4. Go to the Auto-Fill tab in the popup and set the API URL to this server's address.
  5. Navigate to a job posting on Workday, Greenhouse, Lever, BambooHR, iCIMS, SmartRecruiters, LinkedIn Easy Apply, or Taleo.
  6. Click Analyze to get your match score and best-fit skills.
  7. Click Auto-Fill — the extension detects all form fields and fills them using your profile + AI.
  8. Click Cover Letter to generate a personalized letter for that specific company and role.
  9. Click Generate PDF to download a tailored resume matched to the job description.
